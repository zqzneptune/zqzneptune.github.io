Data Processing: Download, Indexing, and Quantification

Qingzhou Zhang (zqzneptune@gmail.com)

2025-04-10

Introduction

This report outlines the initial bioinformatic steps performed to process the raw data for the Single cell RNA sequencing of vascular smooth muscle cells from atherosclerotic lesions of ApoE ko mice study [https://www.ncbi.nlm.nih.gov/bioproject/626450]. It covers:

  1. Downloading the raw sequencing reads (FASTQ files).
  2. Preparing the reference transcriptome.
  3. Building the Salmon index required for quantification.
  4. Running Salmon to quantify transcript abundances for each cell/sample.

The output of this stage is typically a set of quantification files (e.g., quant.sf) per sample, which are then further processed to generate a count matrix (often represented as a SingleCellExperiment object, as detailed in subsequent reports).

1. Raw Data Acquisition

The raw sequencing data (FASTQ files) were obtained from Sequence Read Archive (SRA) database with Accession Number PRJNA626450.

To improve quantification accuracy by accounting for reads mapping ambiguously outside annotated transcripts (common with pre-mRNA content or genomic contamination), decoy sequences representing the rest of the genome were included in the index.

The following Bash commands were used to build the Salmon index:


zcat index/mm_gencode_36/transcripts.fa.gz | grep "^>" | sed 's|[<>,]||g' >index/mm_gencode_36/transcripts.txt
gunzip -c index/mm_gencode_36/genome.fa.gz | grep "^>" | cut -d " " -f1 | sed 's/>//g' > "index/mm_gencode_36/decoys.txt"
cat index/mm_gencode_36/transcripts.fa.gz index/mm_gencode_36/genome.fa.gz >index/mm_gencode_36/gentrome.fa.gz

salmon index -t index/mm_gencode_36/gentrome.fa.gz -d index/mm_gencode_36/decoys.txt -p 8 -i index/mm_gencode_36 --gencode
